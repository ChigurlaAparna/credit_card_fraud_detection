About This System

Advanced Machine Learning for Credit Card Fraud Detection

What is FraudShield?

FraudShield is a comprehensive credit card fraud detection system powered by state-of-the-art machine learning algorithms. It analyzes transaction patterns and provides real-time fraud probability predictions with explainable AI insights.

The system uses the famous Kaggle Credit Card Fraud Detection dataset and trains multiple machine learning models including XGBoost, Random Forest, Gradient Boosting, and Logistic Regression.

Key Features

ML-Powered Detection

Multiple algorithms compared and best model selected automatically

SMOTE Balancing

Handles class imbalance using Synthetic Minority Over-sampling

Explainable AI

SHAP-based explanations for every prediction

SQLite Storage

All predictions stored for analysis and history

Dashboard Analytics

Real-time statistics and fraud trend visualization

Responsive UI

Works on desktop and mobile devices

Technical Details

Machine Learning Models
  • XGBoost - Primary model (best performance)
  • Random Forest - Ensemble of decision trees
  • Gradient Boosting - Sequential boosting
  • Logistic Regression - Baseline linear model
Evaluation Metrics
  • Accuracy - Overall correctness
  • Precision - True fraud among predicted fraud
  • Recall - Fraud caught
  • F1 Score - Harmonic mean of precision/recall
  • ROC-AUC - Classification ability
Dataset
  • Source: Kaggle Credit Card Fraud Detection
  • Samples: 284,807 transactions
  • Fraud Rate: ~0.17%
  • Features: V1-V28 (PCA transformed) + Amount + Time
